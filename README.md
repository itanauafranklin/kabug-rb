# kabug-rb
Repositorie of project with Cucumber and Ruby

## How to execute the project

*need have ruby installed

### Instal bundler 
`
gem install bundler 
`

### instal dependencies 
`
bundle install 
`

### Execute locally 
`
bundle exec cucumber 
`

### Execute on CI server (reposts JSON )
`
bundle exec cucumber -p ci 
`
